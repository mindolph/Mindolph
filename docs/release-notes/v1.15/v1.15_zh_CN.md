# v1.15 发行说明（草稿）

### 新功能

* 自定义AI模型供应商(OpenAI兼容).  
	从这个版本开始，你可以定义自己的模型供应商，只要它提供和OpenAI兼容的API即可，不管是本地的还是在线的，不管是自己部署的还是第三方提供的。自定义一个模型供应商只需要设置一个base URL和api key，添加该供应商提供的模型ID。

* 支持百度千帆的模型.

### 改进

* 智能体可以使用自定义的模型供应商. 

* 允许修改智能体或数据集的名称.

* 添加 Qianfan, Moonshot, ChatGLM 更多的模型.

* 移除 DeepSeek 废弃的模型  


### Bug修复

* 某些 Qwen 型号的最大输出标记不正确。

* 最大输出令牌不适用于 Moonshot api。

* 在某些情况下，模型提供程序对“生成/总结”的设置检查不正确。
