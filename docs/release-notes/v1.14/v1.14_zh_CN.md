# v1.4发行说明（草稿）

### 新功能

* 思维导图
	* 为思维导图的主题节点增加属性面板，可以在浏览思维导图的时候快速的读取和编辑笔记和URI
	* 重新设计了笔记编辑对话框的工具栏按钮，把原来多种样式的按钮统一成小图标按钮。

* 国际化支持
	* 增加国际化多语言的特性。
	* 目前支持英语和简体中文。

### 改进

* 默认关闭AI模型的思维模式。

* 更新 GLM 模型。

* 更新 Qwen 型号 `qwen3.6-flash` 和 `qwen3.6-plus`。

* 更新 Moonshot 模型“kimi-k2.6”。

* 更新 DeepSeek 模型 `deepseek-v4-flash` 和 `deepseek-v4-pro`。

* 增加应用启动统计。    

### Bug修复

* 当未选择嵌入存储类型时，应禁用“测试连接”按钮。

### 依赖升级

* 更新 JavaFX 至 25.0.3 和其他依赖。
